## ⚡ How to Run Locally

1. **Clone the repository**  
```bash
git clone https://github.com/AslamKhanpathan142/portfolio.git
```

2. **Install dependencies**

```bash
cd portfolio
npm install
```

3. **Start the development server**

```bash
npm run dev
```
